## 它到底是什么

NetLimiter 是一款专为 Windows 设计的网络流量控制与监控工具，来自捷克布拉格的 Locktime 软件公司。用大白话说，就是一个让你「管得住每个程序联网行为」的软件。

- **查流量统计**（哪个程序历史上偷偷传了多少数据，一目了然）

## 功能细节值得一聊

**程序限速怎么设？**

在主界面找到对应程序，勾选「UL Limit（上传限速）」或「DL Limit（下载限速）」的复选框，直接点旁边的速度数值编辑就行。动态调整，设好之后实时生效，不用重启软件。

**防火墙规则怎么用？**

在「Info View」里点「Add Rule」，规则类型选「Blocker（拦截器）」，指定拦截上传还是下载，或者整个阻断联网。可以针对程序，也可以针对特定 IP/端口。

**优先级和限速有什么区别？**

优先级是「动态分配」，高优先级的程序自动获得更多带宽，但不会绝对剥夺低优先级程序的网络；限速是「硬上限」，精准但偶尔会掐得太死。日常推荐先试优先级，有特殊需求再上限速。

**配置可以备份迁移吗？**

所有规则和过滤器配置都保存在 `%ProgramData%\Locktime\NetLimiter` 下的 XML 文件里。停掉服务、复制文件、粘贴到新设备，重启服务，配置原封不动全搬过去。换电脑不用重新配置一遍。
